# Chatbot-MIST-Help-Desk
A conversational chatbot using **word2vec** and **cosine similarity** to respond to queries about Military Institute of science and technology(MIST)<br><br>

**METHODOLOGY**<br>
                                        
**Language:** Python 3.8

**Frontend:** Tkinter
